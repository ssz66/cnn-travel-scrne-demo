# CNN Travel Scene Classification & AI Research Agent Demo

## 项目简介

本仓库为本科毕业设计相关的项目展示 Demo，主题为基于卷积神经网络的旅游场景图像分类研究。

本人本科专业为大数据方向，但在毕业设计阶段主动选择人工智能与深度学习相关课题，希望通过一个具有挑战性的项目系统学习 CNN、PyTorch、模型训练与实验分析。研究生阶段导师方向也与深度学习相关，因此本项目不仅服务于本科毕业设计，也会作为后续研究生阶段学习和科研实践的基础。

由于毕业论文仍在进行中，本仓库暂不公开完整论文、完整数据集、完整实验结果和学校内部材料，仅展示部分脱敏后的代码结构、实验流程和项目规划。

## 当前研究内容

本项目主要围绕旅游场景图像分类任务，尝试使用以下卷积神经网络模型进行实验和对比：

- AlexNet
- VGG
- ResNet

实验关注内容包括：

- 图像分类数据集组织
- PyTorch 模型训练流程
- 训练集与验证集划分
- Loss 和 Accuracy 变化趋势
- 不同 CNN 模型的分类效果对比
- 深度学习代码理解与调试

## AI 工具使用经历

在完成毕业设计和深度学习学习的过程中，我主要使用 ChatGPT 和 Codex 辅助完成以下任务：

- 理解 PyTorch 代码结构
- 学习 CNN、AlexNet、VGG、ResNet 等模型
- 修改模型训练脚本
- 分析 CUDA、torchvision、权重下载等报错
- 整理实验结果
- 梳理论文实验分析思路

这些经历让我认识到，大模型不仅可以作为问答工具，也可以进一步通过 API 和 Agent 工作流接入真实的学习、科研和代码实践场景。

## AI Research Agent 规划

后续计划基于大模型 API 和 Agent 工作流，尝试构建一个面向学生科研场景的 AI 辅助工具，主要功能包括：

1. 深度学习代码解释
2. PyTorch 报错分析
3. 实验日志整理
4. 模型对比结果总结
5. 论文实验分析辅助
6. “代码学习—问题诊断—修改建议—实验总结”的半自动工作流

该 Agent 计划服务于本科毕业设计和研究生阶段的深度学习科研实践，帮助初学者更高效地完成模型学习、代码调试和实验总结。
